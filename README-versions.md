# Mundoredes - Versiones del Sitio Web

## 📋 Descripción General

Este proyecto contiene dos versiones del sitio web de Mundoredes:

### 🌐 **Versión Completa** (`/`)
- Sitio web corporativo completo con todas las funcionalidades
- Secciones detalladas: Hero, Servicios, Estándar, Cotizador, Área de Clientes
- Sistema de autenticación completo
- Portal de emergencias 24/7
- Navegación completa con menú móvil

### 🎯 **Versión Simplificada** (`/simple`)
- Versión optimizada para conversión y usabilidad
- Diseño más directo y fácil de navegar
- Enfoque en acciones principales: cotización y contacto
- Modal de cotización rápida
- Estadísticas destacadas
- Testimonios de clientes
- Llamados a la acción prominentes

## 🚀 Cómo Usar

### Acceso a las Versiones

1. **Versión Completa**: `http://localhost:3000/` (ruta por defecto)
2. **Versión Simplificada**: `http://localhost:3000/simple`

### Navegación Entre Versiones

- Desde la versión completa: Botón "Versión Simple" en el navbar
- Desde la versión simplificada: Botón "Versión Completa" en el header

## 🎨 Características de la Versión Simplificada

### ✅ Lo que mantiene:
- **Esencia visual**: Colores industriales (navy, orange), tipografía corporativa
- **Profesionalismo**: Mensajes y tono empresarial
- **Funcionalidades clave**: Sistema de cotización, contacto directo

### 🆕 Lo que mejora:
- **Simplicidad**: Menos secciones, más directo al punto
- **Conversión**: CTAs prominentes, formulario modal accesible
- **Velocidad**: Menos elementos, carga más rápida
- **Movilidad**: Optimizado para dispositivos móviles
- **Claridad**: Mensajes más directos y comprensibles
